## Project Overview

The **ECM platform** is a specialized software ecosystem designed to orchestrate the allocation, tracking, and management of technological assets within a high-density institutional environment. This project addresses the logistical challenges of resource distribution for a population of 7,000 potential users, transitioning from manual processes to a robust, automated digital infrastructure.

This repository currently reflects the progress achieved during the initial research and modeling phases (Workshops 1 and 2).

## Methodological Framework

The development process is governed by a structured engineering methodology that ensures the solution is both scalable and maintainable:

 *   **Systems Development Life Cycle (SDLC):** The project follows a systematic progression from requirement elicitation to logical design, ensuring each phase is validated before technical construction begins.

  *  **Domain-Driven Design (DDD):** The architecture is centered on the core business rules of the lending domain. By isolating these rules from external technical concerns, the system remains adaptable to changing institutional needs.

  *  **Clean Architecture Principles:** A multi-tier approach is employed to achieve a strict separation of concerns. This ensures that the management logic is decoupled from data persistence mechanisms and user interfaces.
